# A blank map

In this example we will create a blank map, just with the basic controls in screen:

```tsx
import { Map } from "ol-kit";
import "ol-kit/dist/index.css";

// Defining the map:
function App() {
  return <Map height={"300px"} width={"100%"}></Map>;
}
```
